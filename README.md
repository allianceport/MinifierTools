
MinifierToolsプラグイン
======================

はじめに
--------

MinifierToolsプラグインを使うと、HTMLのテンプレートをminifyし、出力ファイルサイズを圧縮することができます。

インストール
------------

本プラグインではCPAN ModuleのHTML::Cleanを使用しますので、事前にインストールしてください。
MovableTypeのPluginディレクトリにプラグインのディレクトリを設置してください。
